Backbone Assignment - Tips and Tricks

Although much of a protein backbone assignment may be easy and straight forward, you can often be left with a few difficult sections and odd gaps to fill in due to poor spectral quality or difficult amino acid sequences (e.g. containing amino acid repeats or proline-rich regions). Here are a few tips, tricks and tools to help you finish off those difficult sections.

Comments Box

Use the Comments column in your Backbone Assignment module to make notes on the unassigned residues, e.g. possible assignments or amino acid types.

Navigate from NmrResidues table

Reference Chemical Shifts

If you need a reminder of the standard chemical shifts for amino acids in proteins, open up the Reference Chemical Shift module from the Molecule menu or with RC. This way you can easily check whether the chemical shifts for a particular residue are compatible with certain amino acid types or not.

Residue Information

The Residue Information module is also opened via the Molecule menu. This module provides a list of the occurrences of each amino acid type in the sequence, together with the surrounding residues / motifs in which they appear. You can increase the window for the number of neighbouring residues shown. Assigned residues are shown highlighted in green. This is an easy way to find any remaining un-assigned residues of a particular amino acid type and their neighbouring residues.

Strip Plots from NmrResidues

Sometimes it is helpful to see the remaining unassigned strips together to see if you can identify any links which were not picked up by the program. An easy way to make a strip plot of NmrResidues of your choice, is to use the Strip Plot function. Select the NmrResidues you want to create a strip plot with (e.g. in the Backbone Assignment module or an NmrResidue Table) and go to Spectrum / Make Strip Plot or type SP. Select the Display you want the strips to be shown in, then select use NmrResidue selection and click OK. For more information go to our Creating Strip Plots tutorial.
