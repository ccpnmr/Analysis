Customising Peak Labels

You can type PL or go to right-click / Cycle Peak Lables to cycle through four different peak label display options:

Short: Residue Type, Residue Number and Atom Name, compressed where dimensions are the same, e.g. G28H, N or @92H, N or G28H, N; A27CA
Full: Residue Type, Residue Number and Atom Name for all dimensions, e.g. G28H, G28N or @92H, @92N or G28H, G28N; A27CA
Pid: Project ID (Chain, Residue Number, Residue Type and Atom Name), e.g. A.28.GLY.H, N or @-.@92..H, N or A.28.GLY.H, N; A.ALA.27.CA
Minimal: Residue Type and Number, e.g.G28 or @92 or G28; A27

You can set your preferred default peak labels in Project / Preferences (or type Ctrl/Cmd ,) in the Spectrum tab under Peak Annotations.

Peak symbols can be changed by typing PS or going to right-click / Cycle Peak Symbols. Again there are four options which you can cycle through:

Cross: X
Line widths: these are drawn as closed ellipses if the line width has been determined or dotted ellipses if the line width is unknown.
Filled line widths: there are drawn as filled ellipses if the line width has been determined or flower/star-style ellipses if the line width is unknown.
Plus: +

Again, your preferred default peak symbol can be set in Project / Preferences (or type Ctrl/Cmd ,) in the Spectrum tab under Symbols.

The colour of both the peak labels and the peak symbols of a peak list can be changed (independently) by double-clicking on a peak list in the sidebar to bring up the Peak List Properties dialog.

Peaks can also be hidden/shown by right-clicking on a spectrum button in the Spectrum toolbar. Under Peak Lists you can show/hide one or all of the peak lists associated with that spectrum.
