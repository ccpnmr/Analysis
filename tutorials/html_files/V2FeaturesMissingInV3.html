V2 features missing in V3

There are a number of features in V2.4 which we have not yet implemented in V3. We are working on this, though! Our aim is not only to replicate, but to improve and expand features as we implement them.

Tiles / separators in spectrum displays

Coming soon!

Assignment Graph

V3 does have the Sequence Graph which is similar, but doesn't include as many features. Currently, the Sequence Graph module has several speed issues and we may decide to redesign this completely.

Reference Chemical Shifts

Although you can view these with RC, the graphs are currently not as interactive and do not show a cursor linked to the other spectrum windows.

Synthetic peak lists

It is currently possible to create a synthetic peak list when importing a BMRB file (see BMRB Import), but not from a chemical shift list or pdb file (unless you try writing your own macro).

Structure related features

It is possible to read in pdb files and restraint data from NEF files, but otherwise few structure-related features are implemented as yet.

Isotope Labelling

At the moment we have no plans to include this in V3, as it does not appear to be used much. If you think this is important, then please do contact us to discuss this.
