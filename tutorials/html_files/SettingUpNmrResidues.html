Setting up NmrResidues

And explanation of why to do this

First of all set the experiment type for you base spectrum (usually an 1H-15N HSQC) - go to Setting Experiment Types find out how to do this.

Make sure all the peaks have been picked (go to Peak Picking for more information on how to do this).

Type SN or go to Assignment / Set up NmrResidues. Select the (HSQC) peak list
