Setting up NmrResidues

Setting up your NmrResidues is the first step required for the Backbone Assignment of a protein. We start the assignment process by assigning each peak to NmrResidues and NmrAtoms which we use as placeholders until (over the course of the assignment process) we can assign them to actual residues and atoms. Go to NmrResidues for more information on how NmrResidues, NmrAtoms and NmrChains are used in Analysis.

First of all set the experiment type for your base spectrum (usually a 1H-15N HSQC) - go to Setting Experiment Types find out how to do this.

Make sure all the peaks have been picked (go to Peak Picking for more information on how to do this).

Type SN or go to Assignment / Set up NmrResidues. Select the (HSQC) peak list you want to use to set up your NmrResidues. Select the NmrChain you would like your NmrResidues to be added to. By default this is the @- chain. If you want to use a different NmrChain, you can create a new one, by opening up NmrChains in the sidebar and double-clicking on New NmrChain. Then select Create New (rather than Clone from) and give your NmrChain a name. Once you have selected both your source peak list and your NmrChain you can click Okay to set up your NmrResidues. Each peak will now be labelled with an NmrResidue and with NmrAtoms appropriate to the spectrum you are using (H and N in the case of an HSQC). If you open your NmrChain in the sidebar you will now see that it contains all the NmrResidues and these in turn contain the NmrAtoms that have just been created. These are now your provisional atoms names until you have managed to assign them. Any other peaks in other spectra which stem from the same nuclei can be assigned to these same NmrAtoms.
