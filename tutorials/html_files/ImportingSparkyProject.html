Importing a Sparky Project

Analysis V3 contains a simple Sparky importer. Since Sparky projects don't always follow a completely standard setup, we cannot guarantee that your project will be imported correctly, but hopefully most of your data will be transferred into CcpNmr AnalysisAssign V3 correctly.

You can import a Sparky project by dragging your Sparky .proj file into the Drop Area.

Changing Amino Acid Types from 1-letter to 3-letter code

Amino acid types in Sparky projects are often given in one-letter code. Analysis, by contrast, uses uppercase three-letter code (IUPAC and NEF convention). In order for Analysis to recognise your assignment correctly, your assignments must contain the amino acids in three letter code. We have included a simple macro which will allow you to convert your amino acid names. Go to Macro/Open and open the Convert123LetterAACode.py file in the XXX/YYY directory. Click Run to run the macro.
